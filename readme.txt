####LIVE CHAT INSTALLATION PROCESS####
[youtube https://www.youtube.com/watch?v=DmJ00Drn28s&t=3s&ab_channel=HelpCrunchSquadHelpCrunchSquad]

You don't have to be a mastermind to [add the HelpCrunch live chat on your WordPress-powered website](https://docs.helpcrunch.com/installation/helpcrunch-installation-guide-wordpress). Just follow this step-by-step instruction:

**STEP 1.** Log into your WordPress account.
**STEP 2.** Find the Plugins section and press **Add new**.
**STEP 3.** In the Search box, type in **HelpCrunch**.
**STEP 4.** Hit **Install** and **Activate**.
**STEP 5.** [Log in](https://helpcrunch.com/signin.html) to your HelpCrunch account in a separate browser tab.
**STEP 6.** Go to **Settings â†’ Channels â†’ Website Widgets** and choose the widget you'd like to install on your WordPress website.
**STEP 7.** You will be redirected to the **Installation** page. Select **WordPress installation guide** and copy the code snippet from the HTML box.
**STEP 8.** Go back to your WordPress account and find the HelpCrunch live chat widget under **installed plugins**. Open plugin **Settings**.
**STEP 9.** Paste the copied HelpCrunch code snippet into the **HelpCrunch Code** field and save changes.

Now our live chat widget is displayed in the bottom right corner of your website.

If you have any problems with the setup, please check the [installation guide](https://docs.helpcrunch.com/installation/helpcrunch-installation-guide-wordpress). == Frequently Asked Questions ==
= What is HelpCrunch? =
HelpCrunch is a one-stop platform for creating long-term customer relationships. It mixes such advanced tools as multichannel live chat, chatbot, knowledge base, shared inbox, and a popup builder for lead generation, sales, customer support, and customer engagement.

= Is there a free trial or is HelpCrunch only paid to use? =
Both! You can [sign up for a 14-day free trial](http://helpcrunch.com/signup.html?utm_source=wordpress+plugin&utm_medium=marketplace&utm_campaign=wordpress) with HelpCrunch (no card required) and once you are ready to upgrade, pick a pricing plan that suits your business goals best. The costs start from $15/month per team member.

= The HelpCrunch account is installed. What comes next? =

Congratulations! Now, people can reach out to you via a chat bubble they see on your website without superfluous movements.

= Do I need to install any additional tools to use the HelpCrunch live chat plugin for WordPress? =

No. HelpCrunch is a fully optimized solution for WordPress websites. You can start chatting right after you install the HelpCrunch WordPress chat plugin.

= Can I check if some messages from my customers are still open? =

Yes. All messages and conversations are grouped and available in your shared inbox.

= Is HelpCrunch available for mobile? =

Sure! We have [Chat SDK & Messaging API](https://helpcrunch.com/chat-sdk.html) for iOS and Android that makes in-app messaging a breeze. Additionally, you can install HelpCrunch Inbox iOS or Android app to manage your conversations on the go.

= Does HelpCrunch offer a chatbot? =

Yes! You can install a chatbot from HelpCrunch on your website with no additional code.

= Does the HelpCrunch chatbot proactive? =

For now, our chatbot is reactive. But we're doing our best to make it proactive. To activate the bot, your customers should send a message in the chat widget.

= Is the HelpCrunch account necessary for installing the plugin on a WordPress website? =

Yes. One of the steps in the [step-by-step installation guide](https://docs.helpcrunch.com/installation/helpcrunch-installation-guide-wordpress) says that to install the chat plugin, you need to copy the HelpCrunch code from the settings section in your HelpCrunch account.

= Can I specify the pages where I want to show my WordPress widget? =

Yes. You can show or hide some pages where you want to show the widget.

= What other languages are there in HelpCrunch other than English? =

The chat widget is translated by default into Spanish, Portuguese, French, German, Italian, Dutch, Polish, Ukrainian, Lithuanian, Serbian, and Georgian. You can add as many additional languages as you want.

= Does HelpCrunch offer a WordPress plugin only? =

Of course not! HelpCrunch is so much more with functionalities for customer support, engagement, or sales. Messenger integrations, chatbot, shared inbox, knowledge base, email marketing, auto messages, popups, reports, and mobile live chat apps are in its basic toolset.
